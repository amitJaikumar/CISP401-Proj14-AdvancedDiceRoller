# CISP 401- Project 14- Dice Roller 

## Overview
Java dice rolling program.
The program reads dice notation like XdY and generates random results for each die.
Input Format - XdY
* X equals number of dice
* Y equals number of sides per die

## Program Details
* Reads user input from console
* Splits input into number of dice and number of sides
* Rolls each die using randomness
* Calculates total sum of all rolls


## Input Rules
* Input must follow XdY format
* Invalid input causes error message
* Each die rolls a value between 1 and Y

## Key Ideas 
* MVC structure
* String parsing
* Random number generation
* Array usage and handling
